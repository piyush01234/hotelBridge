# Hotel Bridge 🏨

Hotel Bridge is a Hotel Booking & Management System built using Spring Boot.  
It provides APIs for managing hotels, users, bookings, and supporting secure authentication and authorization.

---
 🚀 Features

- Hotel search and listing management  
- Room booking and reservation system  
- User management (admin & customers)  
- JWT-based authentication & role-based access control  
- Input validations and global exception handling  
- RESTful APIs for all core operations  

---

🛠️ Tech Stack

- Java  
- Spring Boot  
- Spring Security  
- JPA / Hibernate  
- MySQL  
- Maven  

---

🔐 Security

- JWT authentication  
- Role-based access control (Admin / User)  
- Secure API endpoints  

---

📦 Project Structure

- Controller Layer – Handles API requests  
- Service Layer – Business logic  
- Repository Layer – Database operations using JPA  
- Entity Layer – Database models  
- Exception Handling – Centralized error handling
